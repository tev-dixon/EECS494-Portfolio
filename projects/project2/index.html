Color Ascent

A 3D Co-Op first person puzzle platformer; developed for EECS 494 at the University of Michigan.

About the Project

This game was created for the EECS 494 class project at the University of Michigan. This is my second shot at making a Unity game; it was completed in just two weeks while juggling four other courses. The game's primary complexities were: developing a voxel engine in Unity, integrating multiplayer, resolving collisions without relying on Unity colliders (AABB), and creating a streamlined map creation/save/load system.

Technologies Used:

  • Unity Game Engine
  • C# Programming Language
  • MagicaVoxel Software
  • Python Programming Language
  • Unity Version Control (Formerly Plastic SCM)
  • Jira for Project Management
