Color Ascent

A 3D Co-Op first person puzzle platformer; developed for EECS 494 at the University of Michigan.

About the Project

This game was created for the EECS 494 class project at the University of Michigan.

I had no teammates for this project and created everything myself (with the help of the tools and asset libraries listed in the credits.)

The game's primary complexities were: developing the voxel engine in Unity, resolving collisions without relying on Unity colliders, and creating a streamlined map creation/save/load system.

Technologies Used:

  • Unity Game Engine
  • C# Programming Language
  • MagicaVoxel Software
  • Python Programming Language
  • Unity Version Control (Formerly Plastic SCM)
  • Jira for Project Management
