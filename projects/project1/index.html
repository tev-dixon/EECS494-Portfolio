Legend of Zelda Remake

A remake of a classic game with a twist; developed for EECS 494 at the University of Michigan.

About the Project

This game was created for the EECS 494 class project at the University of Michigan. This is my first attempt at making a Unity game; it was completed in just two weeks while juggling four other courses. This game features the classic Legend of Zelda first dungeon along with a brand new game-changing feature: a portal gun.

Technologies Used:

  • Unity Game Engine
  • C# Programming Language
  • Unity Version Control (Formerly Plastic SCM)
  • Jira for Project Management

Game Details

The classic game Legend of Zelda meets the game Portal. Go dungeon crawling with a portal blaster, solve complex puzzles, and enjoy this exciting reimagination of the Legend of Zelda.

Game Controls

  • WASD/Arrow Keys: Movement
  • X: Primary Weapon (Sword)
  • Z: Secondary Weapon

Cheat Codes

  • Press 1: God Mode (infinite resources and unlocks all weapons)
  • Press 4: Go to the portal blaster dungeon
