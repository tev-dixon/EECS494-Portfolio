Legend of Zelda

A remake of a classic game with a twist; developed for EECS 494 at the University of Michigan.

About the Project

This game was created for the EECS 494 class project at the University of Michigan. It was a collaborative effort between two students. I, Thomas, was responsible for developing the enemy AI, stalfos, keese, goriya, the weapons, the portal blaster implementation, the health/damage/knockback systems, doors, collectibles, the inventory system, pushable blocks, loot mechanics, and stability/bug fixes.

Technologies Used:

  • Unity Game Engine
  • C# Programming Language
  • Unity Version Control (Formerly Plastic SCM)
  • Jira for Project Management
  • Discord for Organization

Game Details

The classic game Legend of Zelda meets the game Portal. Go dungeon crawling with a portal blaster, solve complex puzzles, and enjoy this exciting reimagination of the Legend of Zelda.

Game Controls

  • WASD/Arrow Keys: Movement
  • X: Primary Weapon (Sword)
  • Z: Secondary Weapon

Cheat Codes

  • Press 1: God Mode (infinite resources and unlocks all weapons)
  • Press 4: Go to p1_gold dungeon (the portal blaster dungeon)
